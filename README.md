#Gymgorithm
Gymgorithm is a fitness and gym management web application designed to help users track workouts, manage fitness progress, and stay consistent with their fitness goals.

#Description
Gymgorithm allows users to create profiles, log workouts, and monitor their fitness progress over time. It also includes an AI-powered chatbot to assist users with fitness-related questions and guidance. The project focuses on health, discipline, and productivity while integrating modern web technologies.

#Features
- User registration and login system (Firebase Authentication)
- Create and update user profiles
- Workout tracking and progress monitoring
- Fitness dashboard with visual charts
- AI Chatbot for fitness assistance (powered by Ollama)
- Responsive design for mobile and desktop
- Clean and user-friendly interface

#AI Chatbot
Gymgorithm includes an AI chatbot powered by **Ollama**, allowing users to:
- Ask fitness-related questions
- Get workout suggestions
- Receive basic health and training guidance
- Interact with an AI assistant in real-time

#Technologies Used
- HTML
- CSS
- JavaScript
- Firebase (Authentication & Firestore)
- Chart.js (for progress visualization)
- Ollama (AI chatbot integration)
